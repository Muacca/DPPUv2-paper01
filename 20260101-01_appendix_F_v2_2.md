## F.2 動作環境と依存パッケージ

### F.2.1 必要環境

- Python 3.10 以上
- Jupyter Notebook または JupyterLab
- `ipympl` バックエンド（`%matplotlib widget` 用）

### F.2.2 依存パッケージ

主要な依存パッケージ：

| パッケージ | バージョン | 用途 |
|:----------|:---------:|:-----|
| numpy | >= 1.20 | 数値計算 |
| scipy | >= 1.7 | 最適化（Brent法） |
| matplotlib | >= 3.5 | 可視化 |
| sympy | >= 1.10 | 記号計算 |
| ipywidgets | >= 8.0 | インタラクティブUI |
| ipympl | >= 0.9 | Jupyter widget backend |

完全なリストは `requirements.txt` を参照。

### F.2.3 インストール例

```bash
pip install -r requirements.txt
```

JupyterLab を使用する場合は追加で：
```bash
jupyter labextension install @jupyter-widgets/jupyterlab-manager jupyter-matplotlib
```

### F.2.4 実行手順

1. 検証付き計算の実行：
    - 詳細は 付録B を参照
   ```bash
   python DPPUv2_runner_S3S1_v3.py --mode MX --ny-variant FULL
   ```
2. パラメータスキャンの実行：
    - 詳細は 付録C を参照
   ```bash
   python DPPUv2_parameter_scan_v3.py
   ```
3. 相図の一括作成：
   ```bash
   python DPPUv2_visualize_phasemap_v3.py dppu_scan_{topology}_{variant}_{timestamp}.csv
   
   python DPPUv2_visualize_phasematrix_v3.py dppu_scan_all_{timestamp}.csv
   ```
4. 結果の可視化：
    - 詳細は 付録D を参照
   ```bash
   jupyter notebook DPPUv2_visualize_notebook_v3.ipynb
   ```
5. 論文中の図の作成：
   ```bash
   jupyter notebook DPPUv2_Paper_Figures_v3.ipynb
   ```
